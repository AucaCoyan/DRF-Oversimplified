# Installation

1. Clone the repo

```
git clone https://github.com/AucaCoyan/DRF-Oversimplified
```

2. Install dependencies

```
pipenv install
```

3. Enter into the venv

```
pipenv shell
```

4. Run manage.py

```
cd myproject
python manage.py runserver
```
